# unity-2D-game-project
2D game project for Unity at HPC
- 2026年度能開大の学校祭で出展するための2Dゲーム制作プロジェクト
- ゲーム設計，Unityプロジェクト，参考文献などを一元管理する予定

## 2Dゲーム設計
### ゲームテーマ（仮）-->『Hopping Mile』
飛び跳ねるオブジェクトを操作して流れ来る障害物を回避し，高得点を稼ぐ&ゴールを目指す
